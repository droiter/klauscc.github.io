PCA Whitening

2016/7/26 posted in  deep learning

Introduction

  • PCA(Principal Components Analysis),主成分分析，可以用于降维或者whitening(白化)。

PCA

PCA找到一个更低维度的子空间使我们的数据\(x = \{x^{(1)},x^{(2)},…,x^{(m)}\}\)投影上去：

\(x_{rot}=U^Tx\)

其中U为
\(U=\left[ \begin{matrix}|&|&&| \\ u_1 & u_2 &...& u_3 \\ |&|&&|& \end{matrix}\right]\)

\(u_1,u_2...\)为x的协方差矩阵\(\Sigma\)的特征向量, \(\lambda_1,\lambda_2...\)分别为对应的特征值，\(\lambda_1,\lambda_2...\)从大到小排序。

当x的均值为0时，协方差矩阵为

\(\Sigma=\frac{1}{m}\Sigma_{i=1}^{m}(x^{(i)})(x^{(i)})^T\)

Reduce dimension

我们有

\(x_{rot}=U^Tx\) （1）
\(x=Ux_{rot}\) （2）

x为\(n*m\)维矩阵
如果我们将\(x_{rot}\)第k+1到第n维置0.即
\(\tilde{x}=x_{rot},where x_{rot,j}=0, k<j<n\)
\(\hat{x}=U\tilde{x}\approx x\).

保留的方差比例通过下式计算：
\(\frac{\Sigma_{j=1}^{k}\lambda_j}{\Sigma_{j=1}^{n}\lambda_j}\)

通常选取最小的k 满足\(\frac{\Sigma_{j=1}^{k}\lambda_j}{\Sigma_{j=1}^{n}\lambda_j} \ge0.99\)

PCA on Images

在进行PCA之前，通常需要满足两个条件(1)特征的均值为0，（2）不同的特征有相似的方差。对于图片（语音，文本，词向量等）条件（2）都满足，只需要满足（1）：
对于每个特征(比如从图片中截取的16x16的patch)，计算其均值，然后该特征的每个像素减去该均值即可。

Whitening

对于每个i, 0

\(x_{PCAwhite,i}=\frac{x_{rot,i}}{\sqrt{\lambda_i}}\)

\(\lambda_i\) 为特征向量\(u_i\)对于的特征值
白化可以和降维结合使用，即取\(x_{rot}\)的前k维

通常，在i接近n的时候，\(\lambda_i\)很小，\(1/\sqrt{\lambda_i}\)会很大且会导致精度丢失，所以常常加一个小常数\(\epsilon\):

\(x_{PCAwhite,i}=\frac{x_{rot,i}}{\sqrt{\lambda_i}+\epsilon}\)

在\(x\)取值在[-1,1]之间时，通常取\(\epsilon \approx 10^{-5}\)

ZCA Whitenning

若R是正定矩阵，则\(RR^T=R^TR=I\)。
在这里选取R=U。定义

\(x_{ZCAwhite}=Ux_{PCAwhite}\)

PCA Whitening Matlab实现

为了计算\(x_{PCAwhite}\), \(x_{ZCAwhite}\)，需要计算出\(x_{rot}\)，所以需要计算出U和\(\lambda\)
实现思路:
1. 计算x的协方差矩阵\( sigma = 1/size(m,2) * x * x' \)
2. 计算sigma的特征向量U和特征值S: [U,S,V]=svd(sigma)
3. \( x_{rot}=U^T *x \)
4. \(x_{PCAwhite} = 1./sqrt(diag(S)+epsilon) * U^T * x\)
5. \(x_{ZCAwhite} = U * x_{PCAwhite}\)

matlab代码

%%Step 0b: Zero-mean the data (by row)
avg = mean(x,1); %x is a n*m maxtrix, m is the nums of images, n is the dimensions of each image
x = x - repmat(avg,size(x,1),1);

%% Step 1a: Implement PCA to obtain xRot
sigma = x * x' / size(x,2);
[U,S,V] = svd(sigma);
xRot = U' * x;

%% Step 1b: Check your implementation of PCA
covar=cov(xRot);

%% Step 2: Find k(0.99),k2(0.9), the number of components to retain
total=trace(S);
cur=0;
k=1;
k2=0;
while ( cur/total < 0.99)
    if(cur/total > 0.9)
        k2=k;
    end
    cur = cur + S(k,k);
    k = k + 1;
end
%% Step 3: Implement PCA with dimension reduction
xHat = U(:,1:k) * xRot(1:k,:);
xHat90 = U(:,1:k2) * xRot(1:k2,:);

%% Step 4a: Implement PCA with whitening and regularisation
epsilon = 1e-1; 
xPCAwhite = diag(1./sqrt(diag(S) + epsilon)) * U' * x;

%% Step 5: Implement ZCA whitening
xZCAWhite = U * xPCAwhite;
