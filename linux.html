Ubuntu 配置静态ipv6地址

在网上搜了很多ipv6配置教程，然而重启以后并不会生效。解决办法:

vim etc/sysctl.d/10-ipv6-privacy.conf
将内容改为:

net.ipv6.conf.all.use_tempaddr = 0
net.ipv6.conf.default.use_tempaddr = 0
